# 證券種類分類

官方文件對證券種類的分類方式大致歸納如下：    
![分類表](/docs/TSIN-And-IND-CODE.png)  
單純從`SIN`或是`股票代號 (STOCK-NO)`來解決定證券種類並非可行作法，故還需要`漲跌幅度資料 (T30)`中的`產業別代碼(IND-CODE)`來協助處理。 

<br><br>

# T30 漲跌幅度資料檔
臺灣證券交易所（TWSE）在`檔案傳輸子系統`上提供的`T30`檔案主要是`漲跌幅度資料`。  
檔案內容包含當日可交易之股票的開盤價、最高價、最低價、收盤價及漲跌幅等資訊，供證券商和金融機構使用，以進行即時行情傳輸與數據分析。
